# Person 3 — Feature Engineering & Documentation (WEEK - 1)

## 📄 Description of Work

My responsibility was to perform feature engineering on the cleaned hotel bookings dataset and ensure proper documentation of the work completed.The goal was to enhance the dataset by creating meaningful features that improve understanding of customer behavior and support further analysis and modeling.

## 📂 Dataset Used

I worked with the dataset obtained after the second cleaning stage, named cleaned_step2.csv. After completing feature engineering, the final dataset was saved as final_cleaned_hotel_bookings.csv.

## ⚙️ Feature Engineering Performed
* Total Stay Duration :
I created a new feature representing the total length of stay by combining weekday and weekend nights. This provides a clearer measure of how long each guest stayed at the hotel.
* Total Number of Guests :
A new feature was generated by summing adults, children, and babies for each booking. This helps in understanding booking size and guest composition.
* Booking Type Classification :
Bookings were categorized into Early Booking and Last Minute Booking based on the lead time. This feature helps analyze booking behavior patterns and customer planning habits.
* Season Identification :
A seasonal feature was created using the arrival month. Each booking was classified into one of four seasons: Winter, Summer, Monsoon, or Autumn. This enables seasonal trend analysis in hotel bookings.
